ios facebook integration fix! this was a feature first introduced in ios 6, and was removed in ios 11. it allowed you to post text and location to your profile directly from notification center. you could upload photos to facebook directly from the photos app. you could share links from safari. all without having to open the facebook app or even have it installed! after the feature was removed, apple deleted their developer application on facebook, invalidating their access token (facebook's term for an api key). this tweak simply modifies the request bodies to use a different functional access token, as well as rewriting the url for post privacy options, as that was moved sometime last year.
